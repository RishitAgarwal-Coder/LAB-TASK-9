# **Searching Algorithm Lab Tasks**

This repository contains C programs for implementing fundamental searching techniques in data structures.

**1. Linear Search Implementation:**
- A simple sequential search that traverses the entire array from start to finish.
- Compares each element with the target value until a match is found or the list ends.
- Works effectively on both sorted and unsorted datasets.

**2. Binary Search Implementation:**
- An optimized search algorithm that follows the "divide and conquer" principle.
- **Requirement:** The input array must be sorted beforehand to function correctly.
- Repeatedly halves the search space by comparing the target to the middle element, ensuring $O(\log n)$ efficiency.

**Objective:**
Mastering basic search logic and understanding the impact of data organization (sorting) on the efficiency of data retrieval algorithms.
